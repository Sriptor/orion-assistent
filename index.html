Совместим с Windows, Linux и macOS. Бесплатно и с открытым исходным кодом.

Windows Linux macOS Исходный код

Быстрый старт

# Клонируйте репозиторий
git clone https://github.com/Sriptor/orion-assistant.git
cd orion-assistant

# Установите зависимости
pip install -r requirements.txt

# Запустите Орион
python orion.py
